This chapter reviews foundational concepts in real analysis which underpin the more abstract machinery of functional analysis. Even readers with strong mathematical backgrounds will benefit from a concise, structured refresher.

1.1 Limits and Continuity

Definition (Limit): Let \( f: \mathbb{R} \to \mathbb{R} \). We say that \( \lim_{x \to a} f(x) = L \) if for every \( \varepsilon > 0 \), there exists \( \delta > 0 \) such that whenever \( 0 < |x - a| < \delta \), we have \( |f(x) - L| < \varepsilon \).

Continuity is defined as the property that the function value and the limit coincide:

Definition (Continuity): A function \( f \) is continuous at \( x = a \) if \( \lim_{x \to a} f(x) = f(a) \).

1.2 Differentiation and Integration

Differentiation measures local linear change. Integration aggregates values over intervals and extends to Lebesgue integration in more advanced contexts.

Example: The Fundamental Theorem of Calculus states that if \( f \) is continuous on \( [a,b] \), and \( F(x) = \int_a^x f(t)\,dt \), then \( F'(x) = f(x) \).

1.3 Metric Spaces

Definition (Metric Space): A metric space is a pair \( (X, d) \) where \( X \) is a set and \( d: X \times X \to [0, \infty) \) is a function satisfying:
  • \( d(x, y) = 0 \Leftrightarrow x = y \)
  • \( d(x, y) = d(y, x) \)
  • \( d(x, z) \leq d(x, y) + d(y, z) \) (Triangle Inequality)

1.4 Completeness

Definition (Complete Metric Space): A metric space \( (X, d) \) is complete if every Cauchy sequence in \( X \) converges to a limit in \( X \).
Example: \( \mathbb{R} \) with the usual metric \( d(x,y) = |x - y| \) is a complete metric space.

1.5 Sequences and Series of Functions

We distinguish between pointwise and uniform convergence:

Definition (Uniform Convergence): A sequence \( \{f_n\} \) converges uniformly to \( f \) on \( D \) if: \[ \forall \varepsilon > 0, \exists N \text{ such that } n \geq N \Rightarrow \sup_{x \in D} |f_n(x) - f(x)| < \varepsilon. \]

Chapter 2: Normed Vector Spaces

An introduction to vector spaces equipped with a norm.

Definition (Norm): A norm on a vector space \( V \) over \( \mathbb{R} \) or \( \mathbb{C} \) is a function \( \|\cdot\| : V \to [0, \infty) \) that satisfies positivity, homogeneity, and triangle inequality.
Example: \( \mathbb{R}^n \) with \( \|x\|_2 = \sqrt{x_1^2 + \dots + x_n^2} \) is a normed space.

Chapter 3: Banach Spaces

A Banach space is a complete normed vector space.

Theorem (Banach Fixed-Point): Every contraction on a complete metric space has exactly one fixed point.

Applications include solving integral and differential equations.
